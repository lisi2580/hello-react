# hello-react

React starter with ES6, grunt and webpack.

## What you get

- React 0.14
- ES6 & JSX to ES5 via babel
- webpack with react hot loader
- Compile Sass to CSS
- Grunt tasks
- ES6 & JSX validation with ESLint

## Build

- Have [NodeJS](http://nodejs.org/) installed.
- Run these lines:

```bash
npm install -g grunt-cli
npm install
npm start
```
